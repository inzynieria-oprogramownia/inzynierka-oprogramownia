## Uruchomienie

### Wymagania: Node.js, Git

Klonowanie projektu

```bash
  git clone https://github.com/inzynieria-oprogramownia/inzynierka-oprogramownia.git
```

Wejdz do folderu projektu

```bash
  cd projekt
```

Zainstaluj zależności

```bash
  npm install
```

Uruchom serwer

```bash
  npm run dev
```
